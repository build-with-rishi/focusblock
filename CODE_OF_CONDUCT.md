# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, no harassment, no personal
attacks, keep discussion technical. Maintainers may edit or remove comments,
commits, issues, and PRs that violate this, and may ban repeat offenders.

Report unacceptable behavior privately to the repository owner:
**buildwithrishi@gmail.com**. Reports are handled confidentially.
